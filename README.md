# @tokenring-ai/github

GitHub repository search and documentation retrieval for Token Ring.

This package provides:

- `GitHubService` for GitHub API access
- chat tools for repository search, documentation retrieval, and file retrieval
- `/github ...` commands for interactive use

## Commands

- `/github search <query>`
- `/github docs <owner>/<repo>`
- `/github file <owner>/<repo> <path> [ref]`
